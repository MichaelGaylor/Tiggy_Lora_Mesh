Up and Running in Minutes

No IT department required. If you can plug in a USB cable, you can deploy TiggyOpenMesh.

1

Flash the Firmware

One-click firmware upload to your Heltec modules via USB. Takes under two minutes per node.

2

Place Your Nodes

Mount nodes in barns, on posts, or in the yard. Solar panel optional for remote locations.

3

Connect a Gateway

Plug one node into a PC or Raspberry Pi. The gateway GUI auto-discovers every node on the mesh.

4

Build Your Rules

Drag-and-drop Logic Builder: "If gate opens after 10pm, send Telegram alert." No code needed.

Technical Specifications

Enterprise-grade security and performance at a fraction of the cost.

Range (per hop) 1–3 km built-up · 5–10 km line of sight
Mesh Hops Up to 5 hops per message
Network Size 50–80 nodes per gateway
Encryption AES-128-GCM end-to-end
Power 3.3V · ~80 mA active · solar compatible
Frequency 868 MHz (EU) · 915 MHz (US)
Hardware Heltec V3 (~£15) · T-Deck Plus (~£45 with screen)
Sensors I2C, UART, GPIO, BLE beacons, GPS
Gateway Any PC, Raspberry Pi, or Linux server
Licence Open source — free to use, modify, redistribute
